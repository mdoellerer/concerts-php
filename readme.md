
# Concerts API

A simple API written in PHP using the Laravel framework. The idea started from a necessity to control information about the concerts I attended or was planning to attend in the Future.

## Requirements

The API is originally being developed to connect to a Postgre database runnigng in a docker container. Check the .env file for the connection.
